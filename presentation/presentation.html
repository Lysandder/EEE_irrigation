🔮 LOGIC DESIGN

BOOLEAN EXPRESSION
PUMP = A · B · C + C · D
(Auto Mode) + (Manual Override Mode)
RULE 01

AUTO MODE

Pump ON when soil is DRY (A=1), NOT raining (B=0), tank has water (C=1)

RULE 02

MANUAL OVERRIDE

Pump ON when override is active (D=1) AND tank has water (C=1)

GATES

GATE BREAKDOWN

1× NOT gate (invert Rain)
2× AND gates (conditions)
1× OR gate (combine both)

SAFETY

FAIL SAFE

Tank empty (C=0) blocks ALL outputs — no water = no pump, regardless of other signals.

🐍 PYTHON PROTOTYPE

main.py — Smart Irrigation Controller 
def switch_pump(is_soil_dry: bool, is_raining: bool,
               is_tank_full: bool, is_manual: bool) -> bool:
    # PUMP = A·(NOT B)·C + C·D
    if (is_tank_full and is_manual) or \
       (is_soil_dry and (not is_raining) and is_tank_full):
        return True
    return False

# Test all 16 combinations (4 inputs → 2^4 rows)
combinations = [f"{i:04b}" for i in range(16)]

for idx, combo in enumerate(combinations):
    A = bool(int(combo[0]))  # Soil Dry
    B = bool(int(combo[1]))  # Raining
    C = bool(int(combo[2]))  # Tank Full
    D = bool(int(combo[3]))  # Manual
    result = switch_pump(A, B, C, D)
    print(combo, f"({idx})", "Pump ON" if result else "Pump OFF")
